# Apache Struts s2-045

## Objective

The objective of this lab is to demonstrate the use of the Metasploit Framework to identify, exploit, and validate vulnerabilities in a target system. This exercise focuses on understanding how real-world exploits are executed, from initial scanning to gaining access and verifying system compromise in a controlled environment.

## Skills Learned

- Understanding the Metasploit Framework architecture
- Searching and selecting appropriate exploit modules
- Target enumeration and vulnerability validation
- Exploitation of known vulnerabilities
- Payload selection and configuration
- Establishing reverse shells and Meterpreter sessions
- Post-exploitation techniques (system info gathering, privilege escalation basics)
- Session management in Metasploit
- Security impact analysis of exploited systems

## Tools Used

- Metasploit Framework (msfconsole)
- Nmap
- Kali Linux Terminal
- Exploit database / module search
